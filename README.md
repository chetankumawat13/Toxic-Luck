# 🎴 Card Luck Game

A simple and interactive card-based luck game built using **React.js**.
Place your bet, test your luck, and win rewards based on matching cards.

---

## 🚀 Features

* 💰 Bet system ($10, $100, $150)
* 🎴 Random card generation (A, K, Q, J)
* 🧠 Win logic:

  * 3 same cards → **2x reward**
  * 2 same cards → **1.5x reward**
  * No match → **Lose**
* 🔄 Reset functionality
* 🎯 Real-time balance update
* 📖 Help section (How to Play)

---

## 🎮 How to Play

1. Place a bet using available buttons
2. Click **"Check Luck"**
3. 3 random cards will be revealed
4. Win based on combinations:

   * 3 same cards → 2x reward
   * 2 same cards → 1.5x reward
   * No match → You lose
5. Use **Reset** to play again
